# STM32 VL531X 激光测距工程

## 简介

本仓库提供了一个关于STM32与VL531X激光测距模块的工程文件，文件名为`STM32_VL53.zip`。该工程使用STM32F103VET6作为主控芯片进行测试，主要通过I2C接口与VL531X模块进行通信，实现激光测距功能。

## 工程内容

- **主控芯片**：STM32F103VET6
- **激光测距模块**：VL531X
- **通信接口**：I2C
- **引脚配置**：
  - SDA -> PA2
    - SCL -> PA3
      - XShut -> PA4

      ## 使用说明

      1. **硬件连接**：
         - 将VL531X模块的SDA引脚连接到STM32的PA2引脚。
            - 将VL531X模块的SCL引脚连接到STM32的PA3引脚。
               - 将VL531X模块的XShut引脚连接到STM32的PA4引脚。

               2. **软件配置**：
                  - 解压`STM32_VL53.zip`文件，打开工程文件。
                     - 根据工程中的说明文档进行配置和编译。
                        - 将编译好的程序下载到STM32芯片中。

                        3. **运行测试**：
                           - 上电后，系统将通过I2C接口与VL531X模块进行通信，实现激光测距功能。
                              - 测距结果可以通过串口或其他方式输出。

                              ## 注意事项

                              - 本工程未使用中断方式进行数据处理，如有需要，可自行修改代码。
                              - 请确保硬件连接正确，避免因连接错误导致的设备损坏。
